eclipse를 사용한 java project 생성

1. project 생성
	가. new -> java Project
	-Project는 어플리케이션의 이름
	-영문, 숫자 등을 조합하여 만듦

2. package 생성
	가. Project 생성 후
		최초 package 생성
	나. package명
		회사의 Domain name을 
		거꾸로 시작하여 작성
		ex) https://callor.com -> com.callor.hello (뒤의 이름은 패키지명)
		- 3단계 package라고 한다.
	다. 3단계 package를 base package라고 부른다.

3. 문법오류 : syntax error
	- java 문법 규칙을 지키면서 코딩을 하는 도중 대소문자, 빈칸, 세미콜론(;), 괄호(열고, 닫기의 쌍) 등이
	규칙에 맞지 않을 때 eclipse는 자동으로 감지를 한 후 개발자에게 알려준다.
	- 문법 오류가 발생하면 명령어, 명령문에 오류 밑줄이 발생한다.
